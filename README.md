# Kantan Lunch

## Hướng dẫn cài đặt và chạy backend

### Các bước cài đặt

1. Cài đặt [Node.js](https://nodejs.org/en/)

2. Truy cập vào `kantan_lunch_api`
    ```bash
    cd kantan_lunch_api
    ```

3. Tải dependency
    ```bash
    npm install
    ```

### Chạy ứng dụng

1. Chạy ứng dụng ở chế độ production:
    ```bash
    npm start
    ```

2. Chạy ứng dụng ở chế độ development với `nodemon`:
    ```bash
    npm run dev
    ```
